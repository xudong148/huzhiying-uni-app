## 当前基线

- 目标版本：真生产扩张版
- 经营模式：平台自营撮合、平台统一收款结算
- 终端策略：多端并行、渠道能力分层
- 业务优先级：交易履约优先，内容生态服务转化和留存

## 读法建议

1. 想快速理解产品边界、P0/P1/P2、成功标准：先看产品摘要。
2. 想落页面、流程、角色权限、UI 规则：看主规格。
3. 想落状态机、数据结构、接口契约、错误码和审计：看领域与接口规格。
